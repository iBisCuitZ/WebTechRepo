What is Specificity

Specificity คือ การให้ Priority ของแต่ละเอเลเม้นใน html โดยที่ Element ไหนที่ priority สูงที่สุดจะถูกนำมาใช้งาน
Example

จาก ตัวอย่างของเว็ปนี้หน้า index และ Home ใส่ลำดับของ css สลับกัน เมื่อโค้ดเป็นโค้ดเดียวกัน priority เท่ากัน Web Browser จะอิงจาก ำดับการมา"ก่อน-หลัง"โดยที่โค้ดที่พิมทีหลังจะ Overwrite อันเก่าในตำแหน่งเดียวกัน

How to use

ให้เลือกหน้าจากมุมซ้ายบน โดยทั้ง 2 หน้านั้นใช้โค้ด Html ในส่วน "container" เดียวกัน
